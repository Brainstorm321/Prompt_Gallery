How to run (Windows):
1) Extract this zip INTO your existing site folder that already has assets/ and css/ (or let it create css/).
2) In PowerShell: cd to that folder
3) python -m http.server 8000
4) Open: http://localhost:8000/index_updated.html
Note: If you open the html file by double-clicking (file://), Download may not work due to browser security.
